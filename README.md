## Proxmox Icons

A collection of small, optimized PNG icons hosted on GitHub raw URLs for reliable use in Proxmox VE notes and documentation.

## Overview

This repository provides a set of lightweight PNG icons designed to be embedded in Proxmox Virtual Environment (VE) notes and documentation. Each icon is optimized for clarity and performance, making them ideal for use in Proxmox's Note/Description boxes. 

# Images and logos may vary in size, ranging from 50x50 to 100x100 pixels.
